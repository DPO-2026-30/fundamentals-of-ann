# Рубрика-02 (Project). Кейс-задача 2 «CNN для распознавания образов»

10-балльная шкала. Портреты уровней по роли ML Engineer — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | Данные PlantVillage корректно подготовлены (аугментация, разбиение); разработано несколько моделей CNN; применён transfer learning с разными стратегиями заморозки; метрики многоклассовой классификации рассчитаны корректно; сводный отчёт информативен; выводы аргументированы числами. |
| **8** | Все компоненты есть; метрики верны; часть сравнений неполна. |
| **6** | Модели обучены, но transfer learning применён формально или сравнение моделей поверхностное. |
| **4** | Часть моделей не работает или метрики рассчитаны неверно. |
| **0–3** | Решение не воспроизводимо или метрики не интерпретируются. |

## Критерии

| Критерий | Доля | Индикатор |
| --- | --- | --- |
| Предобработка изображений и набор моделей CNN | 30 % | **DL-1.4** |
| Применение transfer learning (заморозка, дообучение) | 25 % | **DL-1.12** |
| Корректность расчёта метрик многоклассовой классификации | 20 % | DL-1.4, DL-1.12 |
| Сводный отчёт и обоснованный выбор моделей | 10 % | DL-1.4, DL-1.12 |
| Понимание на защите | 15 % | DL-1.4, DL-1.12 |
