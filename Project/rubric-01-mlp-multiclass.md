# Рубрика-01 (Project). Кейс-задача 1 «MLP для многоклассовой классификации»

10-балльная шкала. Портреты уровней по роли ML Engineer — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | Данные корректно подготовлены и разделены; разработано несколько моделей MLP с разными гиперпараметрами; метрики многоклассовой классификации рассчитаны корректно (Accuracy, Precision, Recall, F1); сводный отчёт информативен; выводы по выбору моделей аргументированы числами. |
| **8** | Все компоненты есть; метрики верны; выводы обоснованы, но не исчерпывающи. |
| **6** | Модели обучены, но сравнение поверхностное или часть метрик не рассчитана. |
| **4** | Часть моделей не работает или метрики рассчитаны неверно. |
| **0–3** | Решение не воспроизводимо или метрики не интерпретируются. |

## Критерии

| Критерий | Доля |
| --- | --- |
| Предобработка данных и набор моделей MLP | 30 % |
| Корректность расчёта метрик многоклассовой классификации | 35 % |
| Сводный отчёт и обоснованный выбор моделей | 20 % |
| Понимание на защите | 15 % |

## Связь с БРС

Балл (0–10) входит в оценку ПР 5 (раздел «Защита ПР») — [`README.md`](../README.md#итоговая-оценка).
