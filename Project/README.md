# Проектная часть. Кейс-задачи от индустриального партнёра

Две кейс-задачи разработаны якорным индустриальным партнёром — **АО «Россельхозбанк»**. Это интегральные КИМ, проверяющие несколько индикаторов КРМ в комплексной практической задаче. Кейс-задачи выносятся на защиту практических работ:

- кейс-задача 1 → защита **ПР 5**;
- кейс-задача 2 → защита **ПР 10**.

> Папка для проектных заданий именуется `Project` (по конвенции шаблона); подобных папок может быть несколько. В описании КИМ приводятся ссылки на необходимые ресурсы; вспомогательные материалы — в [`attachments/`](attachments/).
