# Лицензия

Рекомендуемая лицензия для образовательных репозиториев — **Creative Commons Attribution 4.0 International (CC BY 4.0)**.
Она разрешает распространение, переработку и коммерческое использование материала при условии указания авторства и ссылки на лицензию.

## Что выбрать

| Лицензия | Когда применять |
| --- | --- |
| **CC BY 4.0** *(рекомендуется)* | Свободное переиспользование с указанием авторства. По умолчанию для методических материалов и ФОС. |
| CC BY-SA 4.0 | Производные материалы должны распространяться на той же лицензии. |
| CC0 1.0 | Передача в общественное достояние (без условий). |

Для программного кода (скрипты проверки, ноутбуки) применяется отдельная лицензия — MIT или Apache 2.0. В этом случае репозиторий содержит два файла:
`LICENSE-CONTENT.md` (CC BY 4.0 — для текстов, методических материалов, КИМ) и
`LICENSE-CODE.md` (MIT/Apache 2.0 — для программного кода).

## Как добавить текст лицензии

1. На странице репозитория на GitHub: **Add file → Create new file**.
2. Имя файла: `LICENSE` или `LICENSE.md`.
3. Нажать **Choose a license template** и выбрать нужную лицензию.
4. Заполнить год и правообладателя (по умолчанию — ФГБОУ ВО РГАУ – МСХА имени К.А. Тимирязева, кафедра статистики и кибернетики).
5. Сохранить (commit).
