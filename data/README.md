# Данные КРМ

Папка содержит исходную книгу компетентностно-ролевой модели (КРМ) версии 3.0.

## Файлы

| Файл | Описание |
| --- | --- |
| [`krm-v3.0.xlsx`](krm-v3.0.xlsx) | Исходная книга КРМ 3.0. **Не редактировать.** |
| [`competency-catalog.csv`](competency-catalog.csv) | Каталог компетенций и индикаторов, формируемых дисциплиной, с предметными дескрипторами освоения. |
| [`role-competency-matrix.csv`](role-competency-matrix.csv) | Машиночитаемая матрица «роль × компетенция × индикатор × уровень» с привязкой к трудовым функциям КРМ. |

## Структура книги `krm-v3.0.xlsx`

Книга содержит 7 листов:

1. **Роли-компетенции наглядно** — соответствия ролей уровням компетенций (Б/С/П/—).
2. **Роли — определения и функции** — определение роли, трудовые действия, фокус компетенции, функция в ИИ-проекте.
3. **Профессиональные** — код, направление, формулировка, трудовые функции, индикаторы, детальные навыки по уровням (Б/С/П), рекомендуемые темы, источники и примеры кейсов.
4. **Опережающие профессиональные**.
5. **Отраслевые** (с колонкой «Домен»).
6. **Универсальные (надпроф.)** — Soft Skills.
7. **Аббревиатуры разные**.
