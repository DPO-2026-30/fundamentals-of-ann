# Ресурсы

Каталог информационных ресурсов, используемых при разработке и применении контрольно-измерительных материалов. Каждый ресурс описан карточкой с метаданными.

## Типы ресурсов

| Папка | Тип |
| --- | --- |
| [`textbooks/`](textbooks/README.md) | Учебники (основная и дополнительная литература) |
| [`papers/`](papers/README.md) | Научные статьи (Белый список / уровень А*) |
| [`test-banks/`](test-banks/README.md) | Банки тестовых вопросов (к зачёту, к защитам ПР, закрытые тесты к ПР) |
| [`problem-banks/`](problem-banks/README.md) | Банки задач и вариантов |
| [`datasets/`](datasets/README.md) | Наборы данных |
| [`benchmarks/`](benchmarks/README.md) | Бенчмарки |
| [`software/python-libs/`](software/python-libs/README.md) | Python-библиотеки |
| [`llm-prompts/`](llm-prompts/README.md) | Промпты для БЯМ и правила анти-чита |
| [`other/`](other/README.md) | Прочее |

## Единая карточка ресурса

| Поле | Описание |
| --- | --- |
| Название | Полное наименование |
| Тип | Датасет / статья / библиотека / и т. д. |
| Аннотация | Краткое описание и назначение |
| Связанные КИМ | Для каких работ используется |
| Доступ | Способ получения (ссылка, регистрация) |
| Лицензия / условия | Условия использования |
| Дата проверки | ГГГГ-ММ-ДД |

## Ограничения

- Защищённые авторским правом материалы (полнотекстовые учебники, статьи) **не размещаются** — только библиографические карточки со ссылками.
- Бинарные файлы > 10 МБ — через Git LFS или внешние ссылки.
