# Банк задач и вариантов

Хранилище постановок задач и параметров вариантов для практических работ. Варианты формируются по номеру студента (закрепляется на первом занятии).

| КИМ | Что варьируется | Диапазон / правило |
| --- | --- | --- |
| ПР 1 | seed генератора матриц | seed = номер варианта |
| ПР 2 | логическая функция (AND/OR/XOR), распределение весов | по варианту |
| ПР 3 | `training_epochs`, `learning_rate` | по варианту |
| ПР 4 | параметры `make_classification` (число признаков, `random_state`) | по варианту |
| ПР 5 | набор собственных изображений цифр | индивидуально |
| ПР 6 | `batch_size` | по варианту |
| ПР 7 | параметры датасета `make_classification` | по варианту |
| ПР 8 | глубина сети (6–8 слоёв) | по варианту |
| ПР 9 | набор исследуемых `lr` | по варианту |
| ПР 10 | число фильтров, размер ядра | по варианту |
| Кейс 1 | варианты предобработки и набора признаков | по варианту |
| Кейс 2 | подмножество классов, аугментация | по варианту |

## Требования к добавлению

Указать проверяемый результат, сложность, входные данные, эталон/способ проверки и параметризацию варианта.
