# Программное обеспечение (Python-библиотеки)

Версии — текущие на момент проведения дисциплины. Источник: РПД, раздел 9.

| Библиотека | Назначение | Команда установки | Лицензия | Связанные КИМ |
| --- | --- | --- | --- | --- |
| **Python** | Базовый язык | — | PSF License | Все КИМ |
| **Anaconda** | Дистрибутив и управление средой | https://www.anaconda.com/ | BSD / коммерческие компоненты | Все КИМ |
| **Jupyter / Spyder** | Среды разработки (ноутбуки, IDE) | `conda install jupyter spyder` | BSD | Все КИМ |
| **numpy** | Матричные операции, тензоры | `pip install numpy` | BSD | ПР 1–6 |
| **matplotlib** | Визуализация (кривые обучения, ROC) | `pip install matplotlib` | PSF-based | ПР 3, 4, 6–9 |
| **scikit-learn** | Датасеты, метрики, утилиты | `pip install scikit-learn` | BSD-3 | ПР 4, 5, 7; Кейс 1 |
| **PyTorch** | Тензоры, autograd, `nn.Module`, GPU | `pip install torch` (CUDA-сборка при наличии GPU) | BSD-style | ПР 7–10; Кейс 2 |
| **torchvision** | Датасеты, модели (ResNet-18), трансформации | `pip install torchvision` | BSD | ПР 10; Кейс 2 |
| **flake8** | Линтинг кода (качество, PEP-8) | `pip install flake8` | MIT | Все КИМ (критерий качества кода) |

## Требования к добавлению

Указать версию, команду установки, назначение, совместимость и лицензию; приложить минимальный пример использования.
