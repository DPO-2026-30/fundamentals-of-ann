# Методические указания преподавателям: ресурсы

## Подбор и применение ресурсов при разработке КИМ

Каталог ресурсов репозитория — [`../../resources/`](../../resources/README.md). Каждый ресурс описан карточкой с метаданными (аннотация, связанные КИМ, доступ, лицензия, дата проверки).

- **Датасеты** ([`../../resources/datasets/`](../../resources/datasets/README.md)): MNIST, CIFAR-10, SVHN, Default of Credit Card Clients, PlantVillage. При формировании варианта можно ограничивать подмножество классов или вводить аугментацию.
- **Программное обеспечение** ([`../../resources/software/python-libs/`](../../resources/software/python-libs/README.md)): numpy, PyTorch, torchvision, scikit-learn, matplotlib, flake8. Версии — актуальные на момент проведения.
- **Научная литература** ([`../../resources/papers/`](../../resources/papers/README.md)): статьи Белого списка / уровня А* из РПД.
- **Учебная литература** ([`../../resources/textbooks/`](../../resources/textbooks/README.md)): основная и дополнительная.
- **Вопросы** ([`../../resources/test-banks/`](../../resources/test-banks/README.md)): 54 вопроса к зачёту и 37 к защитам ПР — основа для билетов и собеседований.

## Использование генеративного ИИ при подготовке материалов

- Генеративные модели полезны для черновиков формулировок, генерации вариантов и подготовки разъяснений.
- **Контроль обязательен**: формулировки должны быть сверены с КРМ 3.0 ([`../../data/krm-v3.0.xlsx`](../../data/krm-v3.0.xlsx)), а задания — на соответствие уровню (Б/С/П).
- Промпты для БЯМ и правила анти-чита — [`../../resources/llm-prompts/`](../../resources/llm-prompts/README.md).

## Обновление ресурсов

- При обновлении версии библиотеки или датасета — обновить карточку ресурса и дату проверки.
- Защищённые авторским правом материалы (полнотекстовые учебники, статьи) в репозитории **не размещаются** — только библиографические карточки со ссылками.
