# Рубрика-03. ПР 3 «Обучение нейронной сети»

10-балльная шкала. Портреты уровней по роли TADS — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | Обновление весов через backprop реализовано корректно; MSE считается верно; сеть обучается (точность растёт с числом эпох); проведено обучение при 10/1000/100000 эпохах и сравнение; выполнено доп. задание (2-4-1 vs 2-2-1); выводы аргументированы. |
| **8** | Обучение работает, MSE корректен; сравнение по числу эпох есть; доп. задание выполнено частично. |
| **6** | Базовое обновление весов есть, но точность нестабильна или сравнение поверхностное. |
| **4** | Код запускается, но обучение не приводит к росту точности; грубые ошибки в backprop. |
| **0–3** | Код не работает или не объясняется. |

## Критерии

| Критерий | Доля |
| --- | --- |
| Корректность backprop и обновления весов | 40 % |
| Корректность расчёта MSE и метода проверки | 25 % |
| Эксперимент (сравнение эпох / архитектур) и выводы | 20 % |
| Понимание на защите | 15 % |

## Связь с БРС

Балл (0–10) входит в раздел «Защита ПР» — [`README.md`](../README.md#итоговая-оценка).
