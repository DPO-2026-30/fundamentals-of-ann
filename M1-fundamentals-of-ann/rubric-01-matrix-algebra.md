# Рубрика-01. ПР 1 «Основы матричной алгебры»

10-балльная шкала оценивания. Балл рубрики входит в БРС (до 10 баллов за ПР). Портреты уровней по роли TADS — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | Все операции (сложение, умножение на число, умножение матриц, транспонирование) реализованы вручную корректно; сверки с `numpy` пройдены на матрицах ≥10×10; студент уверенно объясняет реализацию и границы применимости операций. |
| **8** | Все операции работают; сверки пройдены; мелкие шероховатости в коде студент находит и исправляет сам; объяснение уверенное. |
| **6** | Реализация в целом верна, но с подсказками; сверки пройдены; объяснение поверхностное. |
| **4** | Часть операций реализована неверно или не сверенa с `numpy`; работает только по образцу; объяснение слабое. |
| **0–3** | Код не работает или студент не может объяснить собственное решение. |

## Критерии (вклад в итоговый балл)

| Критерий | Доля |
| --- | --- |
| Корректность ручной реализации операций | 50 % |
| Сверка результатов с `numpy` на матрицах ≥10×10 | 20 % |
| Качество кода (читаемость, структура класса, соблюдение PEP-8) | 15 % |
| Понимание на защите (объяснение, ответы на вопросы) | 15 % |

## Связь с БРС

Итоговый балл за ПР (0–10) суммируется с баллами за остальные ПР (раздел «Защита ПР», максимум 80 баллов), посещение (10) и конференцию (10) — см. [`README.md`](../README.md#4-итоговая-оценка).
