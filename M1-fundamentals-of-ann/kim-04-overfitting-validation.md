# КИМ-04. Практическая работа 4 «Диагностика переобучения и валидация нейронной сети»

## 1. Назначение и формируемые результаты

КИМ формирует навык визуализации процесса обучения для диагностики переобучения и недообучения, подбора гиперпараметров и оценки качества через ROC-AUC. Это ключевая работа по роли Data Analyst.

## 2. Привязка к КРМ

| Компетенция | Индикатор | Уровень | Роль | Трудовая функция |
| --- | --- | --- | --- | --- |
| DL-1 | DL-1.1 | Б | TADS | Техническая реализация / валидация |
| MF-1 | MF-1.2 | С | Data Analyst | Статистический анализ, тестирование гипотез, визуализация |

## 3. Цель работы

Освоить визуализацию процесса обучения нейронной сети для диагностики переобучения.

## 4. Условия проведения

- **Аудитория:** компьютерный класс.
- **ПО:** Python 3.10+, numpy, scikit-learn, matplotlib.
- **Время:** 2 ак. часа.
- **Вариантность:** параметры `make_classification` (число признаков, `random_state`) — по варианту.

## 5. Материалы и ресурсы

- scikit-learn, matplotlib — [`resources/software/python-libs/`](../resources/software/python-libs/README.md).
- Лекция 4 «Диагностика переобучения и валидация нейронной сети».

## 6. Задание

1. Сгенерировать данные через `sklearn.datasets.make_classification` (задача бинарной классификации).
2. Разделить данные на `train` / `validation` / `test`.
3. Реализовать функцию сохранения состояния модели на лучшую эпоху (критерий: максимум `Accuracy` на `validation`; при равенстве — минимум `Loss`).
4. Реализовать функцию визуализации: графики `Loss` и `Accuracy` по эпохам для `train` и `validation`; ROC-кривые и AUC для `train` и `validation` для лучшей эпохи.
5. Реализовать функцию финальных метрик: `Accuracy` для `train` / `validation` / `test`; ROC-AUC для `test`.
6. Построить и обучить несколько MLP, сделать вывод о пере/недообучении.
7. Улучшить результат (темп обучения, число эпох, число нейронов).
8. Реализовать функции для выбора оптимальных параметров (темп обучения, число эпох, число нейронов) с графиками.
9. Построить сеть с оптимальными параметрами.
