# Рубрика-04. ПР 4 «Диагностика переобучения и валидация»

10-балльная шкала. Портреты уровней по ролям TADS / Data Analyst — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | Реализованы все функции (сохранение лучшей модели, визуализация Loss/Accuracy и ROC-AUC, финальные метрики); корректно разделены train/val/test; построено несколько MLP, сделан обоснованный вывод о пере/недообучении; подобраны оптимальные параметры с графиками; построена итоговая сеть с оптимальными параметрами. |
| **8** | Все функции есть; выводы обоснованы; мелкие недочёты в визуализации или анализе. |
| **6** | Базовые метрики и графики построены, но анализ поверхностный или оптимизация параметров не проведена. |
| **4** | Часть функций отсутствует; ROC-AUC не построен или интерпретирован неверно. |
| **0–3** | Метрики рассчитаны с ошибками; выводы не опираются на графики. |

## Критерии

| Критерий | Доля |
| --- | --- |
| Корректность разбиения и функций сохранения/метрик | 30 % |
| Визуализация (Loss/Accuracy, ROC-AUC) | 25 % |
| Анализ пере/недообучения и подбор гиперпараметров | 30 % |
| Понимание на защите | 15 % |

## Связь с БРС

Балл (0–10) входит в раздел «Защита ПР» — [`README.md`](../README.md#4-итоговая-оценка).
