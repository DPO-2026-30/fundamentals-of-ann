# Рубрика-02. ПР 2 «Построение MLP. Подготовка к обучению»

10-балльная шкала. Портреты уровней по роли TADS — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | Класс сети 2-2-1 реализован со всеми требуемыми атрибутами; матрицы AND/OR/XOR корректно подготовлены; прямой проход (включая `H_sigmoid`, `O_sigmoid`) считается верно; метод проверки работает; студент объясняет каждый шаг. |
| **8** | Всё работает; мелкие недочёты (например, в наименовании атрибутов) исправляются самостоятельно; объяснение уверенное. |
| **6** | Базовая структура есть, прямой проход в целом верен, но с подсказками; метод проверки реализован не полностью. |
| **4** | Архитектура описана, но расчёт матриц содержит ошибки или не для всех функций. |
| **0–3** | Класс не работает или не объясняется студентом. |

## Критерии

| Критерий | Доля |
| --- | --- |
| Корректность структуры сети и атрибутов | 35 % |
| Корректность прямого прохода и метода проверки | 35 % |
| Качество кода и документирование | 15 % |
| Понимание на защите | 15 % |

## Связь с БРС

Балл (0–10) входит в раздел «Защита ПР» — [`README.md`](../README.md#итоговая-оценка).
