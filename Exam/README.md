# Промежуточная аттестация. Зачёт с оценкой

Промежуточная аттестация подтверждает освоение дисциплины в целом и **не должна дублировать только один модуль**:  она охватывает все индикаторы КРМ (MF-1.2; DL-1.1, DL-1.2, DL-1.3, DL-1.4, DL-1.12). Папка именуется `Exam` по конвенции шаблона; описательный MD-файл приведён ниже.

## КИМ

| КИМ | Описание | Индикаторы | Рубрика |
| --- | --- | --- | --- |
| [КИМ-01](kim-01-grade-test.md) | Итоговая зачётная работа | MF-1.2; DL-1.1, DL-1.2, DL-1.3, DL-1.4, DL-1.12 | [рубрика-01](rubric-01-grade-test.md) |

## Условия допуска и порог

- Допуск — только при выполнении всех практических работ.
- Зачёт с оценкой получают обучающиеся, набравшие по БРС **не менее 60 баллов**.
- Набравшие менее 60 баллов при условии выполнения всех практических работ пишут **итоговую зачётную работу** по настоящему КИМ.
- Перечень вопросов к зачёту — в [`resources/test-banks/`](../resources/test-banks/README.md).
