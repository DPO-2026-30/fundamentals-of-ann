# Рубрика-01 (Exam). Итоговая зачётная работа

10-балльная шкала; перевод в 5-балльную шкалу зачёта с оценкой — в [`README.md`](../README.md#4-итоговая-оценка).

| Уровень | Описание |
| --- | --- |
| **10** | Полные и точные ответы на все теоретические вопросы; практическая часть решена корректно и оптимално; код чистый; обучающийся уверенно объясняет решение и отвечает на уточняющие вопросы по всем индикаторам. |
| **8** | Ответы верные и полные; практическая часть решена; мелкие недочёты, которые исправляются при собеседовании. |
| **6** | Ответы в целом верные, но с пробелами; практическая часть решена с подсказками. |
| **4** | Часть вопросов не раскрыта; практическая часть содержит ошибки или не доведена до конца. |
| **0–3** | Ответы неверны или практическая часть не решена. |

## Критерии

| Критерий | Доля |
| --- | --- |
| Теоретические ответы (охват всех индикаторов) | 45 % |
| Практическая часть (корректность, чистота кода) | 35 % |
| Собеседование (понимание, способность объяснить) | 20 % |

## Перевод балла в оценку зачёта

| Балл рубрики | Оценка |
| --- | --- |
| 10–9 | Отлично |
| 8–7 | Хорошо |
| 6–6 | Удовлетворительно |
| < 6 | Неудовлетворительно |

> Соответствие процентной шкале БРС и итоговому решению о зачёте — в [`README.md`](../README.md#4-итоговая-оценка).
