# КИМ-09. Практическая работа 9 «Алгоритмы оптимизации нейронной сети»

## 1. Назначение и формируемые результаты

КИМ формирует понимание и навык реализации алгоритмов оптимизации: Momentum SGD, RMSprop, Adam; исследование их чувствительности к скорости обучения.

## 3. Цель работы

Освоить реализацию и сравнение алгоритмов оптимизации нейронных сетей.

## 4. Условия проведения

- **Аудитория:** компьютерный класс.
- **ПО:** Python 3.10+, PyTorch, matplotlib.
- **Время:** 4 ак. часа (2 пары).
- **Вариантность:** набор исследуемых `lr` — по варианту.

## 6. Задание

1. Реализовать Momentum SGD (в собственном модуле или явным циклом на тензорах PyTorch без `torch.optim`), сверить с `torch.optim.SGD(momentum=…)`.
2. Обучить одну модель четырьмя оптимизаторами (SGD, SGD+Momentum, RMSprop, Adam) при одинаковом `lr`, построить кривые Loss.
3. Подобрать оптимальный `lr` для каждого оптимизатора.
4. Исследовать чувствительность «`lr` → качество на validation».
5. Свести результаты в таблицу (эпохи до порога accuracy, итоговое качество, время).
6. **Дополнительно:** работа при разных размерах мини-батча.
