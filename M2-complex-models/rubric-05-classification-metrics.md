# Рубрика-05. ПР 5 «Метрики классификации»

10-балльная шкала. Портреты уровней по роли Data Analyst — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | Сеть обучена на MNIST, проверена на `mnist_test` и на собственных изображениях; функция метрик (Accuracy, Error rate, Precision, Recall, F1, Fβ) реализована вручную как часть модуля ИНС; метрики рассчитаны корректно; выводы аргументированы; выполнено доп. задание (CIFAR-10/SVHN). |
| **8** | Основное задание выполнено, метрики верны; доп. задание частично; мелкие недочёты. |
| **6** | Сеть обучена, но функция метрик реализована не полностью или содержит ошибки. |
| **4** | Часть метрик не рассчитана или интерпретирована неверно. |
| **0–3** | Метрики рассчитаны с ошибками или не на своих данных. |

## Критерии

| Критерий | Доля |
| --- | --- |
| Обучение сети и тестирование (включая собственные изображения) | 30 % |
| Корректность функции метрик многоклассовой классификации | 40 % |
| Выводы по метрикам | 15 % |
| Понимание на защите (включая кейс-задачу 1) | 15 % |

## Связь с БРС

Балл (0–10) входит в раздел «Защита ПР» — [`README.md`](../README.md#итоговая-оценка). Защита ПР 5 включает кейс-задачу 1.
