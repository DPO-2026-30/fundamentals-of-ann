# КИМ-06. Практическая работа 6 «Mini-batch GD. Оценка Latency и Throughput»

## 1. Назначение и формируемые результаты

КИМ формирует навык реализации градиентного спуска по мини-батчам и оценки производительности инференса (Latency, Throughput) — ключевая работа по роли ML Engineer.

## 3. Цель работы

Освоить градиентный спуск по мини-батчам и оценку производительности нейронной сети (Latency, Throughput).

## 4. Условия проведения

- **Аудитория:** компьютерный класс.
- **ПО:** Python 3.10+, numpy, matplotlib, PyTorch.
- **Время:** 2 ак. часа (форма — **практическая подготовка**).
- **Вариантность:** размер мини-батча — по варианту.

## 5. Материалы и ресурсы

- MNIST, CIFAR-10, SVHN — [`resources/datasets/`](../resources/datasets/README.md).

- Лекция 6 «Градиентный спуск по мини-батчам (Mini-batch GD). Оценка Latency и Throughput».

## 6. Задание

1. Реализовать градиентный спуск по мини-батчам для MLP на MNIST при произвольном `batch_size`.
2. Разделить train/validation/test, визуализировать, сделать вывод о пере/недообучении.
3. Построить модели с разными гиперпараметрами, построить графики accuracy.
4. Подобрать оптимальные гиперпараметры (скорость обучения, число эпох, число нейронов, `batch_size`) — в балансе accuracy / Throughput / Latency.
5. Повторить для CIFAR-10, SVHN.
