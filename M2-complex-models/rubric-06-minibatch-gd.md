# Рубрика-06. ПР 6 «Mini-batch GD. Latency и Throughput»

10-балльная шкала. Портреты уровней по роли ML Engineer — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | Mini-batch GD реализован для произвольного `batch_size`; данные корректно разделены и визуализированы; подобраны оптимальные гиперпараметры с балансом accuracy/Throughput/Latency; эксперимент повторён для MNIST, CIFAR-10, SVHN; замеры Latency/Throughput протоколированы и объяснены. |
| **8** | Основное задание выполнено; баланс подобран; повтор для всех трёх датасетов — частично. |
| **6** | Реализация работает, но замеры производительности поверхностные или оптимизация не проведена. |
| **4** | Mini-batch реализован с ошибками; замеры отсутствуют или неверно интерпретированы. |
| **0–3** | Код не работает или результаты не воспроизводимы. |

## Критерии

| Критерий | Доля |
| --- | --- |
| Корректность mini-batch GD и разделения данных | 35 % |
| Подбор гиперпараметров и визуализация | 25 % |
| Замеры и интерпретация Latency/Throughput | 25 % |
| Понимание на защите | 15 % |
