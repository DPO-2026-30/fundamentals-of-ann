# Рубрика-07. ПР 7 «Регуляризация»

10-балльная шкала. Портреты уровней по роли ML Researcher — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | MLP реализован на PyTorch и сопоставлен с собственным модулем; условия переобучения созданы; Model Checkpointing и Early Stopping работают; Weight Decay и Dropout исследованы на всех требуемых значениях; разница `train/eval` продемонстрирована; сводная таблица заполнена корректно; выводы аргументированы. |
| **8** | Все компоненты есть; часть исследований неполна или выводы недостаточно развёрнуты. |
| **6** | Базовая регуляризация применена, но систематического исследования параметров нет. |
| **4** | Часть методов не реализована; режимы `train/eval` не показаны. |
| **0–3** | Эксперименты не воспроизводимы или результаты не корректны. |

## Критерии

| Критерий | Доля |
| --- | --- |
| Реализация MLP на PyTorch и условия переобучения | 25 % |
| Model Checkpointing и Early Stopping | 20 % |
| Исследование Weight Decay и Dropout | 30 % |
| Демонстрация `train/eval` и сводная таблица | 10 % |
| Понимание на защите | 15 % |

## Связь с БРС

Балл (0–10) входит в раздел «Защита ПР» — [`README.md`](../README.md#итоговая-оценка).
