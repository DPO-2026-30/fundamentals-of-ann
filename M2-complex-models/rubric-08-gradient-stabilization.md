# Рубрика-08. ПР 8 «Стабилизация градиентов»

10-балльная шкала. Портреты уровней по роли ML Researcher — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | Глубокий MLP реализован; затухание градиентов продемонстрировано логами нормы по слоям; исследованы `sigmoid`/`tanh`/`ReLU`; инициализации (нулевая, `N(0,1)`, Ксавье, Хе) сравнены; BatchNorm добавлен, режимы training/EMA показаны; общий вывод аргументирован. |
| **8** | Все компоненты есть; отдельные сравнения неполны. |
| **6** | Базовое исследование проведено, но без систематического сравнения инициализаций или без BatchNorm. |
| **4** | Затухание градиентов не продемонстрировано или методы применены неверно. |
| **0–3** | Эксперимент не воспроизводим или выводы не обоснованы. |

## Критерии

| Критерий | Доля |
| --- | --- |
| Демонстрация затухания градиентов (лог нормы) | 25 % |
| Сравнение функций активации | 20 % |
| Исследование инициализаций | 20 % |
| Batch Normalization (включая режимы) | 20 % |
| Понимание на защите | 15 % |

## Связь с БРС

Балл (0–10) входит в раздел «Защита ПР» — [`README.md`](../README.md#итоговая-оценка).
