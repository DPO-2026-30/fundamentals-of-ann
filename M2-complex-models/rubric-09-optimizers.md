# Рубрика-09. ПР 9 «Алгоритмы оптимизации»

10-балльная шкала. Портреты уровней по роли ML Engineer — в [`docs/role-portraits.md`](../docs/role-portraits.md).

| Уровень | Описание |
| --- | --- |
| **10** | Momentum SGD реализован вручную и сверен с `torch.optim.SGD(momentum=…)`; модель обучена четырьмя оптимизаторами при одинаковом `lr`, кривые Loss построены; оптимальный `lr` подобран для каждого; чувствительность исследована; сводная таблица заполнена; выводы аргументированы. |
| **8** | Все компоненты есть; отдельные части (например, доп. исследование по размеру батча) неполны. |
| **6** | Оптимизаторы применены, но без подбора `lr` или без сводной таблицы. |
| **4** | Сверка с эталоном отсутствует или содержит расхождения; выводы слабые. |
| **0–3** | Реализация неверна или результаты не воспроизводимы. |

## Критерии

| Критерий | Доля |
| --- | --- |
| Реализация Momentum SGD и сверка с `torch.optim` | 25 % |
| Сравнение четырёх оптимизаторов (кривые Loss) | 25 % |
| Подбор `lr` и исследование чувствительности | 25 % |
| Сводная таблица и выводы | 10 % |
| Понимание на защите | 15 % |

## Связь с БРС

Балл (0–10) входит в раздел «Защита ПР» — [`README.md`](../README.md#4-итоговая-оценка).
