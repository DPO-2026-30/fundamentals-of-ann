# Модуль 2. Разработка сложных моделей ИНС

**Раздел 2 РПД.** Лекции 5–10 (8 ч), практические работы 5–10 и две кейс-задачи (22 ч, в т.ч. 4 ч практической подготовки — ПР 5, ПР 6). Распознавание образов и метрики классификации; оптимизация обучения (mini-batch GD, регуляризация, стабилизация градиентов, алгоритмы оптимизации); разработка свёрточных нейронных сетей и transfer learning.

**Формируемые результаты:** DL-1.1, DL-1.2, **DL-1.4**, DL-1.12 (Б); MF-1.2 (С).
**Целевые роли:** Data Analyst, ML Engineer, ML Researcher.
