# КИМ-07. Практическая работа 7 «Регуляризация (борьба с переобучением)»

## 1. Назначение и формируемые результаты

КИМ формирует навык применения методов регуляризации (Model Checkpointing, Early Stopping, Weight Decay, Dropout) с использованием PyTorch и понимания режимов `model.train()` / `model.eval()`.

## 3. Цель работы

Освоить методы регуляризации для борьбы с переобучением нейронных сетей.

## 4. Условия проведения

- **Аудитория:** компьютерный класс.

- **Время:** 4 ак. часа (2 пары).
- **Вариантность:** параметры датасета `make_classification` — по варианту.

## 6. Задание

1. Реализовать MLP средствами PyTorch (тензоры, autograd, `nn.Module`), сравнить с собственным модулем из ПР 1–3.
2. Сгенерировать данные (`make_classification`, ≥20 признаков, часть шумовых), разделить на train/validation/test.
3. Создать условия переобучения (2–3 скрытых слоя по 256 нейронов, много эпох).
4. Реализовать Model Checkpointing.
5. Реализовать Early Stopping с `patience`.
6. Исследовать Weight Decay при коэффициентах `0; 1e-4; 1e-2; 1e-1`.
7. Исследовать Dropout при вероятностях `0; 0.2; 0.5; 0.8`.
8. Продемонстрировать разницу `model.train()` / `model.eval()`.
9. Свести результаты в таблицу.
10. **Дополнительно:** повтор на MNIST.
